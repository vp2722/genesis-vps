# Help Command

Display all available GENESIS commands and their usage.

## Usage
```
/help [category]
```

## Categories
- core - Core system commands
- planning - Planning and estimation
- architecture - System design
- development - Implementation
- quality - Testing and review
- deployment - Release and deploy
- operations - Monitoring and incidents
- data - Data operations
- analysis - Risk and impact analysis
- collaboration - Team collaboration
- documentation - Docs generation
- emergency - Emergency procedures

## Output
List all commands in the specified category with:
- Command name
- Brief description
- Usage syntax
- Key agents involved
