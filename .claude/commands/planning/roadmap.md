# Roadmap Command

Display or update the product roadmap.

## Usage
```
/roadmap [view|update]
```

## View
Display current roadmap with:
- Epics and features
- Priority order
- Dependencies
- Status

## Update
With CPO approval:
- Add new items
- Reprioritize
- Update status
- Modify scope
