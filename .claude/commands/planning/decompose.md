# Decompose Command

Break down an epic into implementable user stories.

## Usage
```
/decompose <epic_name>
```

## Process
1. Understand epic scope
2. Identify user journeys
3. Split by vertical slices
4. Create user stories
5. Define dependencies

## Story Sizing
Each story should be:
- Completable in one sprint
- Independently testable
- Delivering user value
