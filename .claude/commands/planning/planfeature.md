# Plan Feature Command

Orchestrate comprehensive feature planning across all agents.

## Usage
```
/planfeature <feature_name>
```

## Process
1. Product Manager: Define requirements and user stories
2. Chief Architect: Design architecture
3. Security Architect: Threat modeling
4. Solutions Architect: Component design
5. Leads: Estimate and plan implementation
6. QA Lead: Test strategy

## Output
Generate feature plan document with:
- User stories with acceptance criteria
- Architecture design
- Security considerations
- Implementation tasks
- Test plan
- Timeline estimate
- Dependencies
- Risks

Save to `docs/features/{feature_name}/plan.md`.
