# Estimate Command

Estimate task complexity with relevant team members.

## Usage
```
/estimate <task_description>
```

## Process
1. Present task to estimators
2. Gather initial estimates
3. Discuss outliers
4. Re-estimate if needed
5. Record consensus

## Fibonacci Scale
- 1: Trivial, <1 hour
- 2: Simple, few hours
- 3: Moderate, half day
- 5: Complex, full day
- 8: Large, 2-3 days
- 13: Very large, split recommended
