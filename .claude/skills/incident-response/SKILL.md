# Incident Response Skill

## Capability
Coordinate incident response and conduct postmortems.

## Incident Process
1. Detection
2. Triage
3. Communication
4. Mitigation
5. Resolution
6. Postmortem

## Severity Levels
- SEV1: Complete outage
- SEV2: Major degradation
- SEV3: Minor degradation
- SEV4: Minimal impact

## Postmortem Template
- Timeline
- Root cause
- Impact
- Response actions
- Lessons learned
- Action items
