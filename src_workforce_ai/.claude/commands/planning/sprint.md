# Sprint Command

Manage sprint lifecycle operations.

## Usage
```
/sprint <action>
```

## Actions
- `plan` - Start sprint planning
- `review` - Conduct sprint review
- `close` - Close sprint and calculate metrics

## Sprint Planning
1. Review backlog with PM
2. Confirm capacity with EM
3. Select stories for sprint
4. Task breakdown
5. Commitment

## Sprint Review
1. Demo completed work
2. Gather feedback
3. Update backlog

## Sprint Close
1. Calculate velocity
2. Update metrics
3. Archive sprint data
4. Prepare for retrospective
