# Architecture Design Skill

## Capability
Design scalable, maintainable system architecture following best practices.

## Design Process
1. Gather requirements (functional & non-functional)
2. Identify constraints
3. Explore options
4. Evaluate trade-offs
5. Document decision (ADR)
6. Create diagrams
7. Review with stakeholders

## Architectural Patterns
- Microservices
- Event-driven
- Hexagonal/Ports & Adapters
- CQRS/Event Sourcing
- Serverless
- Modular Monolith

## Non-Functional Requirements
- Scalability
- Availability
- Performance
- Security
- Maintainability
- Observability
